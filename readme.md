# neovim config

vim ide 打造计划

## 快捷键

### normal 

命令|	功能
----|-------------------
,	|	指令前缀(leader)
jk	|	回退命令模式

### coc

按键|	作用
----|-----------
gd	|	跳转定义
gy	|	跳转引用

### defx

按键	|	作用
--------|-----------
\-		|	打开目录
hjkl	|	类似 coc-explorer
e		|	打开但不进入文件
t		|	open:tab

### leaderF

按键|	作用
----|-----------
C-f	|	搜索指定词
C-p	|	文件搜索

### comment

按键|	作用
----|-----------
,cc	|	注释
