# neovim config

vim ide 打造计划

## 快捷键

### normal 

命令		|	功能
------------|-------------------
`,`			|	指令前缀(leader)
`jk`		|	回退命令模式
`<spave>w`	|	=`<C-w>w`

### airline & buffers

命令			|	功能
----------------|-------------------
`<space>[0-9]`	|	跳转指定标签
`<space>h`		|	上一个标签
`<space>l`		|	下一个标签
`<space>q`		|	关闭标签

### coc

按键	|	作用
--------|-----------
`gd`	|	跳转定义
`gy`	|	跳转引用

### defx

按键		|	作用
------------|-----------
`-`			|	打开目录
`nd`		|	new directory
`nf`		|	new file
`yy`		|	copy file
`p`			|	paste file
`hjkl`		|	浏览目录
`e`			|	打开但不进入文件
`t`			|	open:tab

### leaderF

按键	|	作用
--------|-----------
`C-f`	|	搜索指定词
`C-p`	|	文件搜索

### git

按键	|	作用
--------|-----------
`ghs`	|	?存储git hunk
`ghu`	|	git undo
`ghp`	|	git preview

### others

按键	|	作用
--------|-----------
`,cc`	|	注释
`,t`	|	翻译
