# neovim config

vim ide 打造计划

## 快捷键

### normal 

命令|	功能
----|-------------------
,	|	指令前缀(leader)
jk	|	回退命令模式

### airline & buffers

命令|	功能
----|-------------------
<space>[0-9]	|	跳转指定标签
<spave>h		|	上一个标签
<spave>l		|	下一个标签

### coc

按键|	作用
----|-----------
gd	|	跳转定义
gy	|	跳转引用

### defx

按键	|	作用
--------|-----------
\-		|	打开目录
nd		|	new directory
nf		|	new file
yy		|	copy file
p		|	paste file
hjkl	|	浏览目录
e		|	打开但不进入文件
t		|	open:tab

### leaderF

按键|	作用
----|-----------
C-f	|	搜索指定词
C-p	|	文件搜索

### comment

按键|	作用
----|-----------
,cc	|	注释
